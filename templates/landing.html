ROSSMANN

App web : Prévisions de ventes

Cette application web repose sur des techniques de Machine Learning et de Data Science afin de prédire les ventes futures des magasins Rossmann à partir de données historiques.

Le modèle exploite plusieurs facteurs tels que les promotions, la concurrence, les jours fériés et les tendances saisonnières afin de fournir des estimations fiables et exploitables.
